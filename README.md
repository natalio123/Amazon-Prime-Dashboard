# Amazon Prime Video Content Analysis Dashboard
## Project Overview
Proyek ini bertujuan untuk menganalisis dan memvisualisasikan dataset konten Amazon Prime Video yang mencakup data dari tahun 1920 hingga 2021. Dashboard interaktif ini membantu dalam memahami distribusi konten, preferensi genre, serta segmentasi rating audiens untuk mendukung pengambilan keputusan strategis berbasis data.

## Key Features & Insights
Content Inventory: Menampilkan total inventaris yang terdiri dari 9.655 judul, 5.771 sutradara, dan 519 genre yang berbeda.

Ratings Distribution: Menganalisis klasifikasi konten berdasarkan audiens, dengan temuan utama bahwa rating 13+ merupakan kategori terbesar dengan jumlah lebih dari 2,1 ribu judul.

Genre Trends: Mengidentifikasi Drama dan Komedi sebagai genre dominan di platform Amazon Prime Video.

Content Type Split: Memvisualisasikan perbandingan antara Film (Movies) dan Acara TV (TV Shows), di mana film mendominasi sebesar 80,82% dari total konten.

Historical Timeline: Melacak pertumbuhan volume konten dari waktu ke waktu, menunjukkan peningkatan signifikan pada periode pasca-2000.

## Tech Stack
Tool: Microsoft Power BI Desktop.
Data Source: Amazon Prime Video Dataset (1920 - 2021).

Visualizations: Card visuals, Bar charts, Area charts, Donut charts, dan Slicers.

Data Preparation & Analysis
Data Cleaning: Melakukan pembersihan data pada kolom sutradara dan genre yang memiliki nilai ganda (multi-valued fields).

Exploratory Data Analysis (EDA): Menganalisis hubungan antara tahun rilis dengan pertumbuhan jumlah judul untuk melihat tren industri streaming.

Dashboard Design: Mengimplementasikan prinsip desain UI/UX dengan tema dark mode khas Amazon Prime untuk meningkatkan keterbacaan data.
